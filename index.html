Manual

迷ったときの使い方

  1. 今日やる を見て、まず3件だけ前に進めます。
  2. 止まり で判断待ちや未確定タスクを確認します。
  3. 作る文章 からText Studioで下書きを作ります。
  4. 全体 と Spreadsheet Dashboard でガント、一覧、完了タスクを確認します。

ステータス

タスクは `未着手 / 進行中 / 判断待ち / 完了` の4状態で管理します。完了したタスクは今日やる一覧から外れ、Dashboardのタスク一覧に残ります。

保存

Text Studioで作った文章は、内容を確認してから保存します。保存した文章はこのブラウザのローカルに残ります。

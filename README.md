# BLP_LANDSCAPE COFEE_MVP

LANDSCAPE COFFEE 07 のブランドローンチを進めるための独立MVPです。

## 役割

- スプレッドシート由来のローンチタスクを見やすくする
- 今日進めること、止まっていること、作る文章を分けて表示する
- Instagram、クラファン、note、DM/依頼文の下書きを作る
- タスク状態と下書きをブラウザのローカル保存で保持する

## 使い方

`index.html` をブラウザで開きます。

簡易パスワード:

```text
landscape07
```

このパスワードはMVP用のクライアント側ガードです。強い認証ではありません。

## 画面構成

- 左: フェーズ切り替え、Manual
- 中央上: 注視点、表示モード切り替え
- 中央左: 今日やる / 止まり / 作る文章
- 中央右: Spreadsheet Dashboard
- 右: Text Studio、Tone Memory

## MVP外

- Google Sheetsへの双方向同期
- Notionへの自動反映
- SNS投稿予約
- クラファンサイトへの直接投稿
- 画像/動画生成
- 本格ログイン認証
